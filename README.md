# AnjinFilesTool
### 使用方法: 
一、文件 - 打开 - 选择一个目录<br>
二、选中文件 - 执行操作 - 保存
### 目前具有的功能:
一、查找文件<br>
二、修改文件名<br>
三、批量查找并替换文件名<br>
(1)文本替换：按照文本精确查找<br>
(2)正则表达式替换：按照正则表达式查找，并且可根据分组替换<br>
&emsp;例:<br>
&emsp;将所有pngX，改为jpgX（X为任意字符），替换规则可为：png([\s\S]) -> jpg{1}<br>
&emsp;将所有pngX，改为XpngX（X为任意字符），替换规则可为：png([\s\S]) -> {1}{0}
